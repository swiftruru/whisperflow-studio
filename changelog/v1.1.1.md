# v1.1.1

## Bug fixes
- Fix `FileNotFoundError: 'poetry'` when running the packaged app from Finder/dock — the GUI environment does not inherit the shell PATH, so the bare `poetry` string failed. The resolved absolute poetry path is now passed to `run_cli.py` via the `WHISPERFLOW_POETRY_PATH` environment variable.
- Fix `ENOENT python/config/config.json` in the packaged app — `python/` and `bridge/` are now placed outside the `.asar` archive via `extraResources` so they are both readable and writable at runtime.
- `settings.json` now stored in `app.getPath('userData')` when running as a packaged app.
