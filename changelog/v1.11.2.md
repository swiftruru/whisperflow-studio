# v1.11.2

Small polish release: the macOS menu's `About` item now opens the in-app About tab instead of the system's black native panel, every modal close button uses the same icon style, and the author bio in the About card is rewritten to read warmer.

## Changed

- **macOS `About WhisperFlow Studio` menu opens the in-app About tab.** Previously this menu item used Electron's `{ role: 'about' }`, which pops the OS's stock dark grey panel that ignores app theming. The menu now routes to the existing `menu:open-about` IPC channel that the renderer was already listening on, switching the user to the styled `關於 / About` tab instead. The plumbing was wired all along — only the menu item still pointed at the native role.
- **Changelog viewer close button** redesigned to match the transcript preview modal: SVG X icon inside a 32 × 32 ghost button with hover tint and focus ring, instead of a `✕` glyph in a default browser button. The shared `.modal-close-btn` class is also exposed for future modals.
- **Author bio in the About card** rewritten from `獨立開發者 / 設計師` / `Independent developer / designer` to `相信軟體可以又好用又可愛` / `Software can be both useful and adorable`. The old line was a generic identity tag; the new one expresses a design value that's actually visible in the product (the polish, the i18n care, the on-theme checkboxes).

## Fixed

- **Stray Windows-only label in the transcript preview modal** (zh-TW) — `在檔案總管顯示` is the Windows term for "File Explorer", but macOS users see Finder and Linux users see whatever file manager their distro ships. The button is now `在資料夾中顯示`, matching the platform-neutral label that the queue and history rows have used since [80802d9](https://github.com/swiftruru/whisperflow-studio/commit/80802d9). (Already shipped in the v1.11.1 hotfix; folded into this release for completeness.)

## Migration notes

- No user action required. The macOS About menu change happens automatically; existing keyboard shortcuts (`⌘ ,` / `⌘ Q` / etc.) are unaffected because they live elsewhere in the menu template.
