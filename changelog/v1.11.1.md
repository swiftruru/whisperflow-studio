# v1.11.1

Tiny patch release fixing a stray platform-specific label in the transcript preview modal.

## Fixed

- **`在檔案總管顯示` → `在資料夾中顯示`** in the transcript preview modal footer (zh-TW). "檔案總管" is the Windows term for File Explorer; macOS users see Finder and Linux users see whatever file manager their distro ships. The other "Show in folder" button in the queue / history rows was already platform-neutral after [80802d9](https://github.com/swiftruru/whisperflow-studio/commit/80802d9), this one slipped through when the transcript modal was added.
