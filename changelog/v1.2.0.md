# v1.2.0

## Config Migration
- Fully retire the legacy `config.ini` references and standardize the app on `python/config/config.json`
- Add `python/config/config.example.json` as the tracked config template for local working copies
- Auto-bootstrap missing `config.json` and `settings.json` from tracked templates

## Shared Metadata
- Add `python/config/config.metadata.json` as the shared source for settings UI options, language prompt presets, media extensions, subtitle extensions, and app runtime defaults
- Update Electron main, preload, renderer, and Python helpers to read the same metadata instead of duplicating hardcoded values
- Normalize profile loading so incomplete profile configs inherit missing keys from the tracked template

## Docs
- Rewrite README setup and architecture docs to match the current `config.json` workflow
- Document tracked templates, shared metadata, and packaged-app behavior more accurately
