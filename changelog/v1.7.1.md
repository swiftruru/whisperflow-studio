# v1.7.1

Patch release that fixes the cramped layout of the **System Check** (preflight) rows on narrow panels, especially under the English UI.

## Fixed

- **Preflight row layout** — previously each preflight row was a flex container with the icon, the title/description body, and the action button all on one horizontal line. Under the English UI the button (e.g. "Create environment now", "Install ffmpeg", "Choose folder") was wide enough to squeeze the description into a one-or-two-word-per-line column, which looked visibly broken on standard panel widths. The row is now a CSS grid with two template areas: the icon and the full body text share the first row, and the action button drops onto a second row, right-aligned. The description paragraph can now use the full card width, and long English button labels no longer need to be truncated or capped at 120px inside preflight rows.

## Migration notes

No user action required. Pure CSS change, no settings or data touched.
