# v1.1.0

## New features
- **Transcription history** — last 10 transcribed files (✓ success / ✗ fail) shown in the main panel, persisted to `userData/history.json`
- **Recent directories** — last 5 used directories appear below the directory card for one-click re-selection
- **Scan count badge** — "共 N 個待轉錄" pill shows total pending files after each scan
- **Reveal in Finder** — open the scanned file's parent folder directly from the UI
- **Window state memory** — window size and position restored on next launch
- **Keyboard shortcuts panel** — press `?` to open a reference sheet of all shortcuts

## Bug fixes
- Console filter buttons (All / Error / Warn / OK) now correctly show/hide log lines
- Theme toggle button: icon no longer touches the border edge (padding fix)

## CI/CD
- Added GitHub Actions workflow: creates macOS (.dmg), Windows (.exe), and Linux (.AppImage) builds automatically on every release
