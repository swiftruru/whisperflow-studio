# v1.0.0 — Initial Release

## Core
- Media scan: recursively find the first media file without a subtitle companion
- One-click transcription via faster-whisper-webui CLI (Poetry subprocess)
- Real-time console: stream Python stdout/stderr with timestamps and color-coded log levels
- Settings panel: edit all Whisper parameters in-app (model, language, VAD, prompts, paths)
- Profile switcher: switch between multiple config profiles
- Drag & drop: drag a folder onto the directory card to set the media root

## UX
- Auto-loop mode: Scan → Transcribe → Scan cycles automatically until all files are done
- Next to Transcribe card: shows found file name and path after each scan
- Step guide: onboarding hints shown when no directory is selected
- Dirty state indicator: Save button highlights when settings have unsaved changes
- Settings section collapse: each group is collapsible; state persisted via localStorage
- Toast notifications: success / info / error feedback for every action
- Execution timer: status bar shows elapsed time while a process is running
- Window title: shows `● Running` during transcription

## Console tools
- Log level filters: All / Error / Warn / OK
- Console search (Cmd+F): find text in log; shows match count
- Copy log to clipboard
- Save Log: export to `.txt` file
- Clear log (manual only)
- Auto-scroll lock toggle

## Appearance
- Pastel cream/yellow light theme + warm dark theme
- Persisted via localStorage; defaults to OS preference on first launch

## Keyboard shortcuts
| Shortcut | Action |
|----------|--------|
| `Cmd+S` | Save settings |
| `Cmd+K` | Clear console |
| `Cmd+F` | Open console search |
| `Escape` | Close console search |
