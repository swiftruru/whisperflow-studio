# v1.15.1

Patch release fixing a macOS-only crash dialog (`A JavaScript error occurred in the main process`) that could appear after closing and reopening the app window from the Dock.

## Fixed

- **IPC handlers no longer re-register on `activate`** ([src/main/main.js](src/main/main.js)) — on macOS, closing the window does not quit the app; clicking the Dock icon fires `activate`, which was calling `registerHandlers()` a second time and throwing `Attempted to register a second handler for 'config:read'`. IPC handlers are now registered once at app startup only; `activate` just recreates the window.
